## Example Invocation

```yaml
- hosts: engineering-access
  vars:
    users:
      - username: alice
        groups: [engineering]
        ssh_keys:
          - ssh-ed25519 AAAA... alice@laptop
  roles:
    - role: user_onboarding
```

Or via the wrapper playbook:

```bash
ansible-playbook playbooks/onboard_users.yml -e @users.yml
```
